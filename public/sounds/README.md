# Фоновые звуки

Добавьте в эту папку MP3-файлы:

- `rain.mp3` — дождь (или используйте свой файл, см. useAmbientSound)
- `ocean.mp3` — океан
- `forest.mp3` — лес
- `bell.mp3` — колокольчик для интервалов в таймере медитации
- `gong.mp3` — гонг в конце сессии

Без bell/gong таймер будет работать, но без звуковых сигналов. Бесплатные звуки: [Pixabay](https://pixabay.com/sound-effects/), [Freesound](https://freesound.org/).
