# ZenFlow — Медитация и дыхательные практики

Минималистичное веб-приложение для проведения дыхательных сессий и медитаций.

## Возможности

- **Dashboard** — приветствие по времени суток, статистика минут медитации, карточки практик
- **Аффирмации** — аффирмации на день, неделю, месяц (боковое меню), обновляются автоматически
- **Сессия медитации** — таймер обратного отсчёта, дыхательный круг (4 сек вдох / 4 сек пауза / 4 сек выдох), фоновые звуки (Дождь, Океан, Лес, Тишина)
- **Dark Mode** — переключатель тёмной темы
- **LocalStorage** — сохранение прогресса

## Технологии

- Next.js 14 (App Router)
- Tailwind CSS
- Framer Motion
- Lucide React

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Приложение откроется на [http://localhost:3000](http://localhost:3000).

## Фоновые звуки

Добавьте MP3-файлы в `public/sounds/`:

- `rain.mp3`
- `ocean.mp3`
- `forest.mp3`

Без них кнопки звуков будут работать, но воспроизведения не будет.

## Как обновить приложение (на Vercel)

1. Внесите изменения в код локально.
2. В терминале в папке проекта выполните:
   ```bash
   cd "путь\к\проекту"
   git add .
   git commit -m "Описание изменений"
   git push
   ```
3. Vercel автоматически соберёт и задеплоит новую версию (1–3 минуты).
4. Обновите страницу в браузере или переустановите PWA — будет новая версия.

## Напоминания

- В меню откройте **«Напоминания»**.
- Нажмите **«Разрешить уведомления»**, если браузер запросит доступ.
- Укажите **время** (например 09:00) и **текст** («Пора сделать 3-минутный breathwork») и нажмите **Добавить**.
- Уведомление придёт в выбранное время, **если вкладка с приложением открыта или приложение запущено в фоне** (PWA). Когда вкладка закрыта полностью, браузер не может показать напоминание без серверных Push-уведомлений.
