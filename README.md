# ZenFlow — Медитация и дыхательные практики

Минималистичное веб-приложение для проведения дыхательных сессий и медитаций.

## Возможности

- **Dashboard** — приветствие по времени суток, статистика минут медитации, карточки практик
- **Аффирмации** — AI генерирует аффирмации на день, неделю, месяц (боковое меню)
- **Сессия медитации** — таймер обратного отсчёта, дыхательный круг (4 сек вдох / 4 сек пауза / 4 сек выдох), фоновые звуки (Дождь, Океан, Лес, Тишина)
- **Dark Mode** — переключатель тёмной темы
- **LocalStorage** — сохранение прогресса

## Технологии

- Next.js 14 (App Router)
- Tailwind CSS
- Framer Motion
- Lucide React

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Приложение откроется на [http://localhost:3000](http://localhost:3000).

## AI-аффирмации

Для работы аффирмаций нужен API-ключ OpenAI:

1. Создайте файл `.env.local` в корне проекта
2. Добавьте строку: `OPENAI_API_KEY=sk-ваш-ключ`
3. Ключ получите на [platform.openai.com](https://platform.openai.com/api-keys)

## Фоновые звуки

Добавьте MP3-файлы в `public/sounds/`:

- `rain.mp3`
- `ocean.mp3`
- `forest.mp3`

Без них кнопки звуков будут работать, но воспроизведения не будет.
